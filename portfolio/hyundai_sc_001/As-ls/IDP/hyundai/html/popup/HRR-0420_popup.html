학습계획 수립

e-Catalog에서 검색 버튼을 눌러 e-Catalog 솔루션들을 나의 학습계획에 추가하실 수 있습니다. 단 파일, 이미지, 웹 링크 솔루션은 추가되지 않습니다.
E-catalog에 없는 솔루션은 학습계획 직접 입력 버튼을 눌러 학습 계획을 직접 추가하실 수 있습니다.

※ 수강 신청되거나 마일리지를 부여받은 학습의 경우 계획에서 삭제하실 수 없습니다.
