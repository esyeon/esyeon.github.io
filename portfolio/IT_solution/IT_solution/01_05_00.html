이메일/전자결재(그룹웨어)

DAOU office

  • 그룹웨어를 사용해야 하는 이유?

    • ▪ 업무 생산성 증가
    • ▪ 사용 내용의 Data화
    • ▪ 소통을 통한 협업 강화
    • ▪ 강력한 모바일
    최적비용

    월 2천원 (전자결재포함)

  • 안정적인 이메일

    (20년이상 그룹웨어전념)

  • 전자결재

    (다양한 양식제공, 향후
    ERP와 연계)

  • 사내 자체 메신저

  • 자체 프로젝트 관리

    (Works, ToDO+)

  • 개발사의 안정성

    (설치형, 클라우드설치형)

  • 기업케어 정책

    (DO Care, 관리자 교육,
    24X365 장애지원)

  • 지속적인 업그레이드 지원

  • 강력한 보안기능

    (OTP, MAM, SSL인증서,
    첨부파일 미리보기 등)

솔루션 강점

  • 솔루션 완성도, 시장지배력, 기업 대응력 업계 수위
  • 가격 경쟁력 (메일만 사용1,000원/user, 전자결재포함 2,000원)
  • 메일 반응속도가 최고 수준임
  • 사용자들도 Helpdesk로 직접 전화해서 도움요청 가능
  • Works 라는 작업진척도 tool 유일하게 적용 (매우 유용함)
  • 기업전용 단독 구축형도 가능

신청/수행절차

  1. 기업요청

  2. 기업안내(웹사이트에서 솔루션 확인)

  3. 기존 이메일 데이터 다운로드

  4. 전용 웹사이트를 통해
    가입/결제 후 사용 가입/결제 하러가기
