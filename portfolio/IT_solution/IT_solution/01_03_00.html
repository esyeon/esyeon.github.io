문서중앙화 개념

피해유형 분석

  • 파일서버 사용은 해킹과 바이러스의 최대 위험 요소

  • 유출 관계자

  • 유출 관계자

    1위 : 휴대용 저장장치

    2위 : 핵심인력 스카우트

    3위 : 복사,절취

    4위 : E-mail

    5위 : 스마트폰 카메라

  • 유출 피해 유형

    42%경쟁사로의 유출

    27%주요 기술인력 빼가기

    25%내부직원의 유출

    24%거래관계에서 유출

    (중복 응답)

솔루션 특장점

  • 다양한 보안SW를 하나의 통합된 솔루션으로 대체 (USB복사불가, 이메일 첨부승인, 개인PC다운로드 불가 등)
  • ‘구축 비용 없이, 매월 사용료는 저렴하게
  • 쉬운 적용 : 2시간 교육 후 바로 사용

솔루션 우수성

  • 600개 기업사를
    보유한 국내 1위 제품

  • 랜섬웨어에 안전한
    차별화된 기술력

  • 종이 없는 사무실!
    스마트 오피스 구현

  • 종이문서 및 프린터 사용 감소로
    경제적 효과 달성

주요기능

  • 랜섬웨어 및 바이러스에 안전

    700개 기업사 중 랜섬웨어
    감엽사례 단 한곳도 없음!

  • 모든 문서를 중앙 서버에!

    PC가 사라져도, 고장나도 PC만 있으면 업무 시작!

  • 유출 경로 차단

    인쇄, 캡쳐, 복사 등의 우회적인
    유출 행위 차단!

  • 문서 버전 및 이력관리

    문서수정시 자동 버전 생성
    가능하며 문서의 모든 사용

  • 삭제 문서 복원 기능

    관리자가 ‘복원’만 클릭하면
    휴지통에서 원래 위치로 복구

  • 모바일 승인

    언제, 어디서나 규정에 따라
    모바일로 승인하여 반출

차별화 내용

  • 시스템 우수성

    700개 이상 대기업, 중견기업 사용 중 단 한번도 서버운영 중단이 없다. (이중화) 만일을 대비해 기업 데이터를 별도 백업 받고

  • 경제적 우수성

    중소기업을 위한 최적비용 적용 (70% 수준) 소기업을 위한 최소비용 구간 신설 별도 유지보수 비용이 없다.

신청/수행절차

  1. 기업요청

  2. 전문가 시연
    (컨설팅 안내)

  3. 관리자교육/사용자교육
    (총 2시간)

  4. 데이터 자동이관 및 사용
